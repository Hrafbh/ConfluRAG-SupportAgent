## Summary
Checklist L1 pour qualifier rapidement un problème d’accès à RoutePilot (sans analyse technique profonde).

## Symptoms
- RoutePilot ne charge pas
- Erreur réseau intermittente
- “Timeout” / “site unreachable”

## Checklist (L1)
1) Vérifier si le problème est global :
   - demander si collègues ont le même souci
   - vérifier la page statut (KB-041 si disponible plus tard)
2) Tester un autre site web (Internet OK ?).
3) Changer de réseau (Wi-Fi ↔ 4G hotspot) si possible.
4) Redémarrer navigateur + tester navigation privée.
5) Vérifier VPN/proxy entreprise (KB-035 / KB-036).

## Informations à collecter
- ville/pays, réseau (wifi/4G), navigateur, heure, message exact.

## Verification
Accès OK après action corrective.

## Escalation
Si plusieurs utilisateurs impactés : escalader comme incident connectivité.
