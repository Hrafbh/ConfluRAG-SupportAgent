## Summary
Guide simple pour activer la MFA et réduire les blocages de connexion.

## Resolution steps
1) Se connecter à RoutePilot (web).
2) Ouvrir “Profil / Sécurité”.
3) Activer MFA.
4) Scanner le QR code avec l’application d’authentification.
5) Entrer le code 6 chiffres.
6) Sauvegarder les codes de secours (KB-015).

## Symptômes fréquents
- Code refusé : vérifier heure auto sur téléphone.
- QR code non scannable : augmenter luminosité / zoom / refaire génération.

## Verification
Déconnexion → reconnexion avec code MFA.

## Escalation
Si l’utilisateur n’a plus accès au téléphone : utiliser procédure KB-013 (reset MFA).
