## Summary
Déblocage de compte après tentatives échouées ou politique sécurité.

## Causes fréquentes
- Trop de tentatives login
- Politique MFA non complétée
- Compte désactivé (offboarding ou action admin)

## Resolution steps
1) Vérifier statut du compte (Actif / Désactivé / Bloqué).
2) Si bloqué par tentatives :
   - déverrouiller dans console admin
   - demander reset mot de passe si nécessaire
3) Si MFA requis : guider vers l’enrôlement (KB-012).
4) Confirmer au user de se reconnecter après 2 minutes.

## Verification
Login OK + accès aux modules selon rôle.

## Escalation
Si blocage récurrent : ouvrir ticket L3 avec horodatage, IP (si disponible) et message d’erreur.
