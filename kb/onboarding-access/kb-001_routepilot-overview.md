## Summary
RoutePilot est un SaaS B2B de pilotage d’opérations de livraison (planification, exécution, preuve de livraison, reporting). Cette page cadre le périmètre fonctionnel et les rôles pour accélérer le support L1/L2.

## Modules (vue simple)
- Planification : création tournées, affectation chauffeurs.
- Exécution : statuts (en cours/livré/échec), incidents terrain.
- POD : signature/photo/scan comme preuve.
- Reporting : retards, taux réussite, exports CSV/PDF.
- Administration : utilisateurs, rôles, paramètres tenant.

## Rôles standard
- Admin : paramétrage tenant, gestion utilisateurs, droits.
- Manager : suivi KPI, validation, reporting.
- Dispatcher : création/ajustement tournées, support terrain.
- Driver : exécution via mobile, POD.
- Viewer : lecture seule.

## Escalation
Si demande hors périmètre (bug produit, incohérence data), ouvrir ticket L3 avec captures + contexte tenant.
