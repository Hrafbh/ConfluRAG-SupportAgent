## Summary
Le scan est utilisé pour identifier colis/étiquettes. Cette page couvre les problèmes fréquents sans entrer dans des détails techniques.

## Symptoms
- Scan ne détecte rien
- Code mal lu / erreur d’identifiant
- Caméra floue / focus instable

## Resolution steps
1) Vérifier permission caméra (KB-029).
2) Nettoyer la lentille.
3) Augmenter luminosité et stabiliser le téléphone.
4) Éloigner/rapprocher légèrement (focus).
5) Tester un autre code-barres (pour exclure un code endommagé).
6) Saisir manuellement si option disponible (fallback).

## Verification
Le code est reconnu et l’action attendue est exécutée (validation colis/étape).

## Escalation
Si un lot d’étiquettes est illisible : escalader Ops (problème qualité impression/format).
